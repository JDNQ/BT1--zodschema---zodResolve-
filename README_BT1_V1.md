# BT1 - Auth (Next.js + NestJS)

Repo hiện sẽ được gộp chung thành **một** dự án (tên thư mục dự án vẫn là `BT1`).

## Cấu trúc đề xuất (sau khi tạo)

- `frontend/`: Next.js 14 (App Router)
- `backend/`: NestJS (Prisma + JWT + Swagger)

## Mục tiêu Phần 1

- Login + Signup dùng React Hook Form + Zod
- Kết nối API tới NestJS
- Lưu JWT vào localStorage và redirect
- Swagger hiển thị request/response đầy đủ
